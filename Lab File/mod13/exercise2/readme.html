SL275: Module13: GUI-Based Applications

Exercise #2: Create Dual Code for the Calculator

(Level 3)

Objective

In this exercise, you will convert the Calculator GUI to be dual-purpose: existing both as a stand-alone application and as an applet.

Directions

Start by coping the CalculatorGUI class file into your working directory from the events exercise directory.

Split the original code into three class files: CalculatorGUI, CalcApp, and CalcApplet. Compile and run the application using the java interpreter and the applet using appletviewer.
