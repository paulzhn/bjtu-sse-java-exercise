SL275: Module14: Threads

Exercise #2: Use Threads in Applet Animation

(Level 2)

Objective

In this exercise you will use a thread to control the animation within an applet.

Directions

  1. Create an applet called ThreadedAnimation that takes 10 images of Duke waving (in the graphics/Duke directory) and displays them in a sequence that makes it look like Duke is waving.

  2. Use the MediaTracker class to smooth the loading of these images. The following code will help: 
    // Instance variable
Image[] images = new Image[10];

// Somewhere in the init() method
MediaTracker mt = new MediaTracker(this);
for(int x = 0; x < images.length; x++) {
  String imageFile = "graphics/Duke/T"+(x+1)+".gif";
  images[x] = getImage(getDocumentBase(),imageFile);
  mt.addImage(images[x],0); // Add image to group 0
}
mt.waitForID(0); // blocks until all images in group 0 are fully loaded

  3. Allow the user to stop and start the animation using successive mouse clicks.
